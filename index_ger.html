Meine Projekte

"jumpeR" ist ein Mobile-Spiel, welches ich mithilfe der Unity-Engine programmiert habe.
Das Spielprinzip ist einfach - Sie steuern ein Würfel und müssen solange sie es schaffen Hindernisse ausweichen.
Die Entwicklungszeit erstreckt sich über ca. 1 Monat. Den Quellcode des Spieles können Sie auf meiner Github Seite hier finden.

"Projekt-Euler" ist eine Reihe von mathematischen Problemen, die man mithife von einer frei auswählbaren Programmiersprache lösen kann. (Mehr Informationen dazu können Sie auf der offiziellen Internetseite hier finden)
Meine Lösungen können sie auf meiner Github seite hier finden. Ich habe dazu die Programmiersprachen C++/C#/Java benutzt.
